### C语言的强制转换与默认转换

##### 2.| 1. 不同类型的变量(值与变量)赋值时，会发生类型转换


#### 强制类型转换
##### 2.| 如何进行强制类型转换？
##### 4.| type name = (type) var 
解释： type-》将要强制转换为什么变量
##### 5.|
```C

{
    int a = 50000;
```
